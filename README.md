## 核心算法流程 (The 5-Step Algorithm)
根据“熵减系统”方法论，所有社交节点必须经过以下审计：

1. **Question Requirements (质疑需求)**：该社交目标是否具有真实价值？
2. **Delete (剔除冗余)**：删除无意义的寒暄（如：早晚安、在吗）。
3. **Simplify (简化路径)**：将复杂的暧昧对话简化为确定的指令/邀约。
4. **Accelerate (加速反馈)**：缩短线上互动时间，快速进行压力测试。
5. **Automate (自动化/SOP化)**：将成功经验转化为可复制的社交脚本。
